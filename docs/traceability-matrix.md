# Requirements Traceability Matrix (RTM)

Project: Bank Chatbot Support System

|Requirement ID|User Story| Use Case| Acceptance Criteria| Priority| Status |
|----------------|------------|----------|---------------------|----------|--------|
| US-001 | Ask Banking Questions | Answer Customer Banking Inquiries | Chatbot provides correct answers to customer banking questions. | Must Have | Planned |
| US-002 | Check Account Balance | View Account Balance | Chatbot displays the correct account balance after successful customer authentication. | Must Have | Planned |
| US-003 | Report Lost ATM Card | Report Lost or Stolen ATM Card | Chatbot confirms the report and initiates the card-blocking process. | Must Have | Planned |
| US-004 | Transfer Customer to Human Agent | Escalate Customer to Support Agent | Chatbot transfers unresolved conversations to a bank support agent. | Should Have | Planned |
| US-005 | Update Chatbot Knowledge Base | Manage Chatbot Knowledge Base | Updated chatbot knowledge is available for future customer interactions. | Could Have | Planned |
